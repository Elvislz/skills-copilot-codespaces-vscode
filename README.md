# Code with GitHub Copilot

_GitHub Copilot can help you code by offering autocomplete-style suggestions right in VS Code and Codespaces._

## Welcome

This repository contains exercises for learning GitHub Copilot in VS Code and Codespaces. You'll learn how to:

- Set up Copilot inside a Codespace or VS Code
- Use Copilot to accept suggested code
- Use Copilot's hub for alternate suggestions  
- Leverage comments to have Copilot auto-suggest code

## Getting Started

1. Open this repository in GitHub Codespaces or clone it to your local machine
2. Make sure you have GitHub Copilot enabled for your account
3. Open the repository in VS Code
4. Start working through the exercises by creating and editing the JavaScript files

## Exercises

The course includes creating the following files:
- `comments.js` - Learn to use Copilot with comments
- `member.js` - Practice with function suggestions
- `skills.js` - Work with calculations and logic
